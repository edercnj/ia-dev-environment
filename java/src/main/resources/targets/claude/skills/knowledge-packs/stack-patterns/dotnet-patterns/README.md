# dotnet-patterns

> .NET 8+-specific patterns: built-in DI, Entity Framework Core, Minimal APIs, IOptions, WebApplicationFactory testing, NativeAOT, ProblemDetails error handling.

| | |
|---|---|
| **Category** | Stack Pattern |

> **Full content**: See [SKILL.md](./SKILL.md) for the complete pattern reference.

## Key Patterns

- Dependency injection with IServiceCollection and constructor injection
- Entity Framework Core with DbContext, migrations, and query patterns
- Web APIs with Minimal APIs and controller-based endpoints
- Configuration with IOptions, IOptionsSnapshot, and IOptionsMonitor
- Testing with WebApplicationFactory and integration test patterns
- NativeAOT constraints and source generators
