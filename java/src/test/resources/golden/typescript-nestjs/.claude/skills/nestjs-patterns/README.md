- Data access with Prisma and TypeORM integration
- Web/HTTP with Controllers, Guards, Interceptors, and Pipes
- Configuration with @nestjs/config and ConfigService
- Testing with Testing module, mocking providers, and e2e tests
- Build and deployment with Docker multi-stage builds
