## Key Patterns

- Spring dependency injection with constructor injection and bean scopes
- @ConfigurationProperties for typed configuration binding
- Spring Data JPA with repository interfaces and query methods
- @RestController and @ControllerAdvice for endpoints and error handling
- Spring AOT, native build support, and configuration profiles
- Health checks with Actuator and transaction management
