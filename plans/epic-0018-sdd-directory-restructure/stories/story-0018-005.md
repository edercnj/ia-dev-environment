## 7. CORRECT vs INCORRECT

```bash
# Migrar ADRs
git mv docs/adr/_TEMPLATE-ADR.md adr/
git mv docs/adr/ADR-0001-intentional-architectural-deviations-for-cli-tool.md adr/

git mv docs/audits/codebase-audit-2026-03-20.md results/audits/
```

```bash

# NÃO criar conteúdo em contracts/ — esta story é apenas scaffolding

# NÃO modificar o conteúdo dos ADRs

```
