## 3. Descrição

Retrofit `x-release` (release orchestrator, 10+ phases) para emitir `TaskCreate` por phase com approval gate integration (EPIC-0043 interactive gates).

### 3.1 Fases (10+)

- Phase 0: Version resolution
- Phase 1: Release branch creation
- Phase 2: Validation (tests, coverage, security)
- Phase 3: CHANGELOG update
- Phase 4: Version bump
- Phase 5: Release branch commit
- Phase 6: PR to main creation
- Phase 7: **Approval gate** (3-option interactive: PROCEED / FIX-PR / ABORT)
- Phase 8: Tag creation
- Phase 9: Back-merge to develop
- Phase 10: Cleanup

Cada phase: TaskCreate + gate PRE/POST

### 3.2 Approval Gate Integration

Phase 7 approval gate (EPIC-0043) integra naturalmente:
- TaskCreate(subject: "release › Phase 7 › Approval gate")
- Interactive gate renderiza task list
- User selects PROCEED → TaskUpdate(phase-7-task, status: completed)

## 3.5 Entrega de Valor

Operador consegue ver cada step de release (version → validation → approval → tag → back-merge).
