## 3. Descrição

Como **desenvolvedor**, eu quero que os 15 core skills tenham nomes padronizados no padrão `x-{categoria}-{ação}` para que eu consiga descobrir skills por autocompletar.

### 3.2 Operações por Skill

Para cada skill na tabela:
1. `git mv` do diretório (preserva histórico)
2. Atualizar campo `name:` no frontmatter do SKILL.md
3. Renomear arquivo correspondente em `targets/github-copilot/skills/`

### 3.3 Entrega de Valor

- **Valor Principal:** 15 core skills com nomes previsíveis
- **Métrica de Sucesso:** Zero diretórios core com prefixo incorreto
- **Impacto no Negócio:** Discoverability imediata via tab-completion
