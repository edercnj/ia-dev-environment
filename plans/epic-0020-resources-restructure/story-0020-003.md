## 3. Descricao

Como **desenvolvedor do ia-dev-environment**, eu quero que todos os recursos especificos do GitHub Copilot estejam agrupados sob `targets/github-copilot/`, para que a convencao de naming `github-*-templates` seja substituida por uma hierarquia clara e consistente.

### Contexto

Atualmente existem 6 pastas com prefixo `github-` espalhadas no top-level: `github-agents-templates/`, `github-skills-templates/`, `github-hooks-templates/`, `github-instructions-templates/`, `github-prompts-templates/`, `github-pr-issue-templates/`. O prefixo e verboso e inconsistente com a falta de prefixo dos recursos Claude.

### 3.1 Movimentacao de diretorios

| Origem | Destino |
| :--- | :--- |
| `github-agents-templates/` | `targets/github-copilot/agents/` |
| `github-skills-templates/` | `targets/github-copilot/skills/` |
| `github-hooks-templates/` | `targets/github-copilot/hooks/` |
| `github-instructions-templates/` | `targets/github-copilot/instructions/` |
| `github-prompts-templates/` | `targets/github-copilot/prompts/` |
| `github-pr-issue-templates/` | `targets/github-copilot/pr-issue-templates/` |

### 3.2 Assemblers a atualizar

- `GithubAgentsAssembler` — probe `"github-agents-templates"` → `"targets/github-copilot/agents"`
- `GithubSkillsAssembler` — probe `"github-skills-templates"` → `"targets/github-copilot/skills"`
- `GithubHooksAssembler` — probe `"github-hooks-templates"` → `"targets/github-copilot/hooks"`
- `GithubInstructionsAssembler` — probe `"github-instructions-templates"` → `"targets/github-copilot/instructions"`
- `GithubPromptsAssembler` — probe `"github-prompts-templates"` → `"targets/github-copilot/prompts"`
- `PrIssueTemplateAssembler` — probe `"github-pr-issue-templates"` → `"targets/github-copilot/pr-issue-templates"`
- `GithubAgentRenderer`, `ContextualInstructionsAssembler` — refs internas

## 3.5 Entrega de Valor

- **Valor Principal:** Hierarquia clara para recursos Copilot, sem prefixo verboso github-*
- **Metrica de Sucesso:** Todos os testes de assemblers GitHub passando com paths novos
- **Impacto no Negocio:** Paridade de organizacao entre targets, reducao de confusao ao navegar recursos
